Alerts

An alert component is used to display a short message in a way that attracts the user's attention without interrupting the user's task.
Some of the use cases are after any Add/Edit operation to show successful message.

Example of Alerts:

Success toaster: Data fetched successfully...!
Error toaster: Somthing went wrong please try again...!
Warning toaster: Somthing went wrong please try again...!
